UK bank account sort code plugin

Simple jQuery plugin that extends a sort code field (normally 6 characters) to three 2 character fields which only allow numerics and auto-tab when completed.

The plugin keeps your original underlying field intact and updated from user input. There is therefore no need for any rework to existing form handlers.

It is free for use - please download and host your own version, we provide no guarantee of availability on this site. This software is provided free and comes with absolutely NO WARRANTY.

By id

No options passed - textboxes have default width of 20px

$(document).ready(function() { $('#sc').sortCode(); });

By class

Textboxes slightly widened with boxwidth property

$(document).ready(function() { $('.sc').sortCode({boxwidth:22}); });

By class - debug on

$(document).ready(function() { $('.sc-db').sortCode({boxwidth:22,debug:true}); });

Debug option reveals the underlying field so you can check it is being updated correctly. It also shows errors encountered at runtime ( there are some deliberate errors in the html below to demonstrate).
